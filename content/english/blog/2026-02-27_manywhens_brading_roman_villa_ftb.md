One of the settings we've been considering for Monty + The AnyWhens shows are food and performance matinées, so we're delighted to be able to announce a mixed songs'n'stories set at [Brading Roman Villa](https://bradingromanvilla.org.uk), with added tapas. (This was originally pitched as cream teas — sweet or savoury — but it seems there's been a menu change... *Let us know which you'd prefer if we get to run this again...*)

__[Tickets available here](https://bradingromanvilla.org.uk/whats-on/live-music-at-the-villa-the-anywhens-14-march/) for *Brading Roman Villa, Saturday, March 14th, 2026, 3.30-5.45pm, tickets £10.00 per person, £16 with tapas plates*.__
